Estructuras de Control

Bucle while

Definición: El bucle while se usa para repetir un bloque de código mientras una condición sea verdadera.

Sintaxis básica:

while(condición) {
  // instrucciones
}

Ejemplo: Contar del 1 al 10

int i = 1;
while(i <= 10) {
  System.out.println(i);
  i++;
}
Diagrama de flujo: Inicio → [Condición i ≤ 10] → Sí → Imprimir i → i++ → Volver a condición ↓ No Fin

Bucle do-while

Definición: Similar a while, pero garantiza que el bloque se ejecute al menos una vez.

Sintaxis:

do {
  // instrucciones
} while(condición);

Ejemplo: Menú simple que repite hasta elegir salir

int opcion;
do {
  System.out.println("1. Saludar");
  System.out.println("2. Salir");
  opcion = scanner.nextInt();
} while(opcion != 2);

Bucle for

Definición: Ideal para iteraciones con número conocido de repeticiones.

Sintaxis:

for(inicialización; condición; incremento) {
  // instrucciones
}

Ejemplo: Recorrer un arreglo e imprimir elementos

int[] arr = {1,2,3,4};
for(int i = 0; i < arr.length; i++) {
  System.out.println(arr[i]);
}

Clases, Objetos y Métodos

Clases y Objetos

Una clase es el molde que define atributos y comportamientos (métodos). Un objeto es una instancia concreta de esa clase.

Analogía: La clase es la "receta" y el objeto es la "galleta" hecha con esa receta.

public class Coche {
  String color;
  String marca;
  int velocidad;
  
  public void acelerar() {
    velocidad += 10;
  }
}

Coche miCoche = new Coche();
miCoche.color = "Rojo";
miCoche.acelerar();

Métodos

Los métodos representan las acciones o comportamientos que puede hacer un objeto.

Sobre carga de métodos (Overloading)

Mismo nombre, diferentes parámetros para mayor flexibilidad.

public class Calculadora {
  int sumar(int a, int b) {
    return a + b;
  }
  double sumar(double a, double b) {
    return a + b;
  }
}

Modificador static

Se usa para atributos y métodos que pertenecen a la clase y no a un objeto específico.

public class Contador {
  static int cuenta = 0;
  
  public Contador() {
    cuenta++;
  }
  
  static int getCuenta() {
    return cuenta;
  }
}

Clases Locales

Son clases definidas dentro de un método, usadas para tareas muy específicas y con alcance limitado.

void metodo() {
  class Local {
    void mostrar() {
      System.out.println("Clase local");
    }
  }
  Local l = new Local();
  l.mostrar();
}

Pilares y Conceptos Avanzados

Encapsulamiento

Protección de los datos ocultándolos con modificadores private y usando métodos get y set para accederlos.

public class Persona {
  private String nombre;

  public String getNombre() {
    return nombre;
  }

  public void setNombre(String nombre) {
    this.nombre = nombre;
  }
}

Composición y Delegación

La composición es cuando una clase incluye objetos de otras clases como atributos.

La delegación consiste en que un objeto “delegue” tareas a otro para no hacer todo él mismo.

class Motor {
  void arrancar() {
    System.out.println("Motor arrancado");
  }
}

class Coche {
  Motor motor = new Motor();

  void encender() {
    motor.arrancar(); // delegación
  }
}

Clases Envoltorio (Wrapper)

Son clases que envuelven tipos primitivos para tratarlos como objetos y usar métodos especiales.

Integer numero = 10;
Double decimal = 15.5;

int n = numero.intValue();
double d = decimal.doubleValue();
