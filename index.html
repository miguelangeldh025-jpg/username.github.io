Estructuras de Control

Bucle while

El bucle while repite un bloque de código mientras la condición sea verdadera.

int i = 1;
while(i <= 10) {
  System.out.println(i);
  i++;
}

Bucle for

El bucle for es ideal para repeticiones con número conocido.

for(int i = 0; i < 5; i++) {
  System.out.println(i);
}
